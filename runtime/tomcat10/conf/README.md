## Usage of the template files
### context.xml.template
The emm.sh generates the context.xml from the context.xml.template file after starting or restarting the EMM.

### server.xml.template
The EST/OST automatically copies the server.xml.template when starting the Tool, as well as after finishing a runtime update.

### server.xml.apr.template
Was used in the past for installations utilizing tomcat-native. Since that was removed from EST/OST, there is no use for it anymore.
